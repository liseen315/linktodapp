# linktodapp
*A blockchain dapp data analysis platform*

## Installation

*mysql*
```mysql
create database linktodapp character set utf8;
```
*python*
```
$ git clone git@github.com:liseen315/linktodapp-backend.git
$ cd inktodapp-backend
$ pipenv install --dev
$ pipenv shell
$ flask forge
$ flask run
* Running on http://127.0.0.1:5000/
```
